# ✈️ Travelio

Travelio is a web application that helps users discover unusually cheap flight deals and generate AI-powered travel itineraries.

## Inspiration
One of our members loves to travel spontaneously and would sometimes book trips without a planned destination if the flight was cheap. However, most travel websites require users to enter a destination before searching. Travelio was built to help users discover unexpected destinations through affordable flight deals.

## What It Does
Given the user's origin city, Travelio displays the best flight deals relative to the average price throughout the month. Additionally, Travelio can generate AI-powered itinerary plans for the user.

## How We Built It
Our tech stack includes:
- Amadeus Travel API (flight data)
- React (frontend)
- Python (backend)
- Gemini API (AI itinerary generation)
- JavaScript
- HTML
- CSS

## Challenges
Our major challenge was that the request for flight data took a long time.

## Accomplishments
We were able to build a working tool that we find useful. Through this project, we learned a lot about building software and using new tools such as APIs.

## What We Learned
We learned how to collaborate effectively as a team and how to work with new tools and APIs.
